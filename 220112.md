VSCode는 통합개발환경(IDE) 중 하나

IDE 의 다른 예로는 파이참이 있음. 파이참은 알고리즘 때 사용.



### Markdown

HTML의 Markup보다 간편한 형태.

단순 텍스트보다는 예쁘게, 각 잡고 뭘 작성하는 것보다는 간편한 편집



1. 알아둬야 할 문법

   - 제목

   ```

   ```

     ```
     *이탤릭체*
     ```
